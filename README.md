# Aplicaciones MFC

Los proyectos de este repositorio son distintas aplicaciones que he desarrollado para aprender
la programación en MFC y Visual C++ para Windows.

La solución fue creada en Visual Studio 2022.

## Proyectos

### Fundamentos_Inicio

Muestra el código mínimo necesario para crear una aplicación de MFC sin el asistente de VS.

Solamente arranca la aplicación y crea una ventana vacía típica de Windows.

### Fundamentos_Scroll

Aplicación obtenida del libro *Programing Windows with MFC* del autor
Jeff Prosise que muestra los conceptos básicos del uso del Scroll y el dibujo en una ventana.
